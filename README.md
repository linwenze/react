## Memos

> 技术架构：案例react+react-router-dom+react-redux+less+ES6+antds-mobile+webpack实现;

说明
### 功能说明

- 使用antds-mobile ui框架,屏宽以750px设计利用postCSS处理等比收缩
- 兼容手机端(iPhone6及以上)
- 支持路由切换
